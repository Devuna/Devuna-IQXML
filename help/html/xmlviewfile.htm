XML:ViewFile

Parameters:

STRING

File Name, document name or program name



Return Type:

Does not return a value

 

Send a file name to this function and the system will open the file with the assigned Windows application.  Your XML file type should be set as a Windows registered file type to XMLSpy or Internet Explorer.

 

 

 ! opens with what is assigned to TXT files (like notepad, UltraEdit, etc.)

 xml:ViewFile('mytext.txt')        

 

 ! opens with what is assigned to XML files

 xml:ViewFile('myxml.xml')        

 

 ! opens mail client

 xml:ViewFile('mailto:sales@mycompany.com')        

 

 ! opens web browser

 xml:ViewFile('http://www.paresi.net')
