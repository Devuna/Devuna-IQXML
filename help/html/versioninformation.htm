Version Information

 

April 8, 2005 - Initial Release.

 

· iQ-XML released as a "free-ware" application provided that the functions, DLL's and all material is not changed.

 

 

Version 3.00 - April 2015

 

· Corrected small bug with XML:Extract function where if the first parameter included the text parameter, it would return blank.  ie:   xml:Extract('src="', '"', HTMLDOC ...)
· Added XML:SetBoundry function
· Cleaned up example program to use XML:SetBoundry
· Cleaned up XML:DebugQueue screens to make them look nicer and more polished.

 

 

Version 3.01 - May 28, 2015

 

· LoadFromFile and LoadFromString now works with JSON documents.  XML and JSON is interchangeably and all parser functions will handle both XML and JSON in the same manner.  (JSON documents all use a standard <root> base element)

 

 

Version 3.02 - June 14, 2015

 

· XML:RootFromString and XML:RootFromFile added  - makes it easier to branch a proper parser routine, or know when an XML transmission has completed versus wait for more data in an HTTP process.
· XML:AddChild now writes faster.

 

 

Version 4.00 - January 2016

 

· Added 35 SA: (Swiss Army) functions for general purpose programming
· Added XML:IsValid to return true if the XML file is a valid XML file.

 

 

Version 4.02 - May 2016

 

· When exporting XML:DebugMyQueue (contents of a queue) to a text file, the format pictures were not being used.  
· Added additional code to be sure the names of the fields being created were in the proper XML name format.  If the field names were incorrect (ie: started with a symbol or a number), they would cause Clarion to GPF when mapping XML field names to the Queue (xml:LoadQueue)
· Added XML:DumpMemory function
· Small corrections to some SA: functions

 

 

Version 4.04.01 - September 12, 2016

 

· Clarion 10 release. This is the final version published by Paresi; source code available on ClarionHub.

 

 

Version 4.04.02 - July 16, 2017

 

· Rebuild of Clarion 10 release from ClarionHub source code.
· Added and revised help file source; added MIT License notice to all source files; revised install program
· Source code is available in the Devuna-IQXML public repository on GitHub.
