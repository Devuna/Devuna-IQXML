Write - Clarion Process Chain

You can use the Clarion Process Template to write out records to an XML file very easily.   You can also easily do this with a file relationship as well.

 

First, for DATA you'd setup a group for each row in the process.  For example, if you are going to write out Header and Detail records because of a Parent/Child relationship, you would have two groups.  In this example, we'll assume we have two files in the process that have a relationship of AccountNo.

 

Using the Clarion Process Template, in the [FILES] section, you would setup your DETAIL and HEADER file as you normally would.

 

Then, for [DATA], you would add a field for each field you want to extract from your file record layout.  These fields can be in any order and do not have to be every field from your record.

 

 

HeaderGroup GROUP

AccountNo string(20),NAME('Account')

Lastname string(20),NAME('LastName')

Firstname string(20),NAME('FirstName')

END

 

DetailGroup GROUP

TranNumber string(20),NAME('Invoice')

TranDate string(20),NAME('Date')

TranAmount string(20),NAME('Amount')

END

 

EMBED: Start of Process:

 

IF NOT XML:CreateXMLFile('myfile.xml')

 XML:CreateParent('root');XML:AddParent()

END

 

 

EMBED End of Process:

 

XML:CloseXMLFile(True)

 

 

EMBED "Lookup Related Records":

 

if LastAccount <> HED:Account

 if once = True

         xml:CloseParent()

 else

         Once = True

 end

 

 HeaderGroup :=: HED:RECORD

 XML:AddFromGroup(HeaderGroup,'Header',True)

 LastAccount = HED:Account

end

 

EMBED Activity for Each Records:

 

DetailGroup :=: DET:RECORD

XML:AddFromGroup(DetailGroup,'Detail',True,True)
