Glossary

Here is a few terms you may need to know when working with iQ-XML.

 

Element Name (Tag)

<ThisIsAnElementName>

End Tag

</ThisIsAnEndTag>

Attribute Name

<ThisIsAnElementName AttributeName=Value>


Node Pointer

The current "FindNextNode" settings being used by iQ-XML.  It is the current "section" of the XML document which is being used as the current section for processing.

 

You can open an XML document using the iQ-XML Examples program (File / Open XML document) and then use the FindNextNode function to find a node.  The XMLDebugger will highlight the current node for you in red.
 

Current Record

The current record that iQ-XML is pointing last read/used.

Parent Node

The element of a higher tag.

Sibling Node

The element on the same level.

Child Node

The element of a lower level.
