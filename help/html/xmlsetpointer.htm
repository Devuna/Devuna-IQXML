XML:SetPointer

Parameters:

LONG

Positive value containing cursor location



Return Type:

SHORT

XML Error Code (see end of document)

 

Sets the cursor location within the XML document.  If there is an existing node pointer, it is cleared and set to an unset state.

 

NOTE:  This function (XML:SetPointer) will not set the current node state for subsequent XML:GotoParent, XML:GotoChild and XML:GotoSibling processing. For full restoration of node state and pointer, it is highly recommend to see and use XML:SaveState() and XML:RestoreState() when moving around the XML document if you wish for the Node Pointer to be retained.

 

See also XML:GetPointer

 

Example:

 

SavePointer =  XML:GetPointer

 

! do some sort of processing on next record

If ~xml:ReadNextRecord(MyQueue,MyData)

 Do ProcessingRoutine

End

                                 

! Go back to the original record (but node pointer is at the next record because of above read)

If XML:SetPointer(SavePointer)

 Message('Error occurred!','Error!')

End

 IF XML:SetPointer(xml:GetPointer()).
