XML:GetTime

Parameters:

(None)



Return Type:

STRING

Time it took between Creating and Closing XML channel.

 

Returns the tie it took to create the XML structure.

 

This would be exactly between XML:CreateXMLFILE() and XML:CloseXMLFile() or XML:CreateXMLString() and XML:CloseXMLString()

 

Note: This function returns text in English.

 

 IF NOT XML:CreateXMLFile('Work.XML',,,True)

 

       XML:CreateParent('BenchMark')

       XML:CreateAttribute('Date',format(today(),@d010-))

       XML:CreateAttribute('Time',format(clock(),@t07))

       XML:AddParent()

 

       xml:createParent('Detail')

       XML:CreateAttribute('Date',format(today(),@d010-))

       XML:CreateAttribute('Time',format(clock(),@t07))

       XML:AddParent()

       loop j = 1 to 60000

          xml:AddElement('Line','Information for Line: ' & J)

       end

       xml:CloseParent()

       xml:CloseParent()

 

       xml:CloseXMLFile()

 

       message('The XML file (Work.XML) completed in ' & XML:GetTime(),'0.55 is a normal Bench Mark',icon:asterisk)

 

   END

 

 

The full format of the string would be "x hours and x mins xx:xx seconds" with the hours and mins removed if not applicable.
