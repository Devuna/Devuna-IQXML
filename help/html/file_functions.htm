File Functions

 

·SA:FixAsFileName - Take a STRING and convert it into a valid File Name (Take a company name and make sure it is valid file name)
 
·SA:FormatFileSize - Take a size of a file (LONG or ULONG) and then format it nicely such as 4MB or 6.2MB or 10K or 8,000 bytes
 
·SA:GetFileInfo - Get information about a file such as the size, date, time, extension, directory, or Version (EXE/DLL) information.
 
·SA:WriteToFile - Take some string text or the contents of a variable and dump it to disk - with or without Date/Time stamp.  Good for debugging.
