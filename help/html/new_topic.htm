XML:ConvertTimeStamptoLocal

Parameters:

STRING

ISO8601 formatted Time String (ie: YYYY-MM-DDtHH:MM:SSxxxx+06:00)

*LONG

Clarion Long to receive Date (Clarion format)

*LONG

Clarion Long to receive Time (Clarion format)

BYTE

True to just convert to GMT/Zulu time rather than local workstation time.



Return Type:

(None)

 

Takes a ISO8601 Date/Time stamp and returns the Date/Time (Clarion Format) in either Local Workstation Date/Time or in GMT/Zulu time.

 

NOTE: The Date is just used for return and does not actually take into consideration the actual date and how it was in or out of DST (Daylight Savings Time).   It is used simply to take a UTC date/time stamp coming into a web service at the present time and knowing what the actual date/time is if needed.   Additional DST adjustments must be made if the date is in a different period.

 

 YYYY-MM-DDTHH:MM:SS(zone adjustment from GMT)

 

 2009-06-30T16:10:18-04:00

 

 

Example:

 

 

ClarionDate        LONG

ClarionTime        LONG

 

 CODE

 

 XML:ConvertTimeStampToLocal('2011-01-01T08:09:30+08:00',ClarionDate,ClarionTime,False)

 

 message('The workstation time is ' & format(ClarionDate,@d017) & ' at ' & Format(ClarionTime,@t07))
