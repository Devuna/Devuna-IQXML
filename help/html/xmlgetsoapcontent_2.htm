XML:GetRawContent

Parameters:

STRING

Entire node


*CSTRING

CSTRING Variable to hold value being returned



Return Type:

BYTE

True if node found / False if not successful

 

Used to quickly retrieve the first value of an element which matches the requested full node.  This allows quick access to the first occurrence of a particular node without having to use the FindNextNode or LoadQueue functions.

 

The Node must be in the form of Node[.Node.Node] starting with the root.    If you are trying to extract the contents of a soap header/envelope, it is best to use the XML:GetSoapContent as it is more efficient.

 

See also XML:GetSoapContent

 

Example:        

 

 

<?xml version="1.0"?>

<catalog>

  <book id="bk101">

     <author>Gambardella, Matthew</author>

     <title>XML Developer's Guide</title>

     <genre>Computer</genre>

     <price>44.95</price>

     <publish_date>2000-10-01</publish_date>

     <description>An in-depth look at creating applications with XML.</description>

  </book>

</catalog>

 

 

! Retrieve the BookID and Author

 

BookID                CSTRING(32)

BookAuthor                CSTRING(32)

 

if not xml:LoadXMLFile('FileAbove.XML')

 

 if xml:GetRawContent('catalog.book.id',BookID).

 if xml:GetRawContent('catalog.book.author',BookAuthor).

 

 message('The book id is ' & BOOKID & ' and the author is ' & BookAuthor,'XML Information',icon:asterisk)

 

end

 

 

The XML:GetRawContent is good for reading single or maybe two bits of information.   The following would produce the same result and is definitely better for getting 2 or more bits of information:

 

 

BookQueue                QUEUE,PRE(BK)

ID                        STRING(32)

Author                STRING(32)

 END

 

if not xml:LoadXMLFile('FileAbove.XML')

 If not XML:FindNextNode('book')

         if xml:LoadQueue(BookQueue,True,False)

         end
