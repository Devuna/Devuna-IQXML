XML:DebugMyQueue

Parameters:

*QUEUE

Any Queue Structure

<STRING>

Window Title



Return Type:

Does not return a value

 

Shows the contents of the passed queue for debug purposes.  This allows you to show the contents of any queue you have in your Clarion application, even for non-XML debugging.

 

Example:        

 

IF NOT XML:LoadQueue(MyQueue)                          

 XML:DebugMyQueue(MyQueue,'My Queue')

END
