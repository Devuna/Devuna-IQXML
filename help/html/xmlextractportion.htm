XML:ExtractPortion

Parameters:

STRING

String to Search

STRING

Text (indicator) to find

STRING

Which portion to extract (R=Right, L=Left)
 

R = Return right portion or blank if not found

L = Return left portion or blank if not found

r = Return right portion or entire string if not found

l = Return left portion or entire string if not found
 

 

Return Type:

 

STRING

 

Text found

 

Extracts the portion of the string (either right or left) of the find text indicated.  The search is based on the first occurrence and is a simple function as opposed to XML:Extract

 

The String To Search and the Text (indicator) to find can be any length, up to 4MB.

 

Portion to Extract must be R,r,L,l only.

 

See Also:  SA:ExtractBetween and SA:ExtractParm

 

 

Example:

 

 

RESULTSTRING                STRING(128)

 

 CODE

 

 RESULTSTRING = XML:ExtractPortion('Robert|James','|','R')   ! result string will be James

 

 RESULTSTRING = XML:ExtractPortion('Robert|James','*','R')   ! result string will be blank

 

 RESULTSTRING = XML:ExtractPortion('Robert|James','|','r')   ! result string will be Robert|James
