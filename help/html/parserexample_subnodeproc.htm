Parser - Sub Node Processing

Below is a full example of processing an XML document with different levels.   First, a sample XML document is shown that will be processed.   The next page shows a Clarion application which will read a header and fill a queue, read the invoices filling another queue, and then process the data.  Once the program processes that set of data, it moves on to the next set.

Here is the sample iQ-XML Clarion code to process the above XML file …

 

CustomerQueue

QUEUE

GivenName

STRING(20)

Surname

STRING(20)

AddressLine

STRING(20),DIM(3)

CityName

STRING(20)

PostalCode

STRING(20)

StateProv

STRING(20)

 End


InvoiceQueue QUEUE

InvoiceNo

STRING(20)

InvoiceDate

STRING(20)

InvoiceAmt

STRING(20)

 End


CRec

SHORT

IRecs

SHORT

 

CODE

 

! Load the XML file to process it

! -------------------------------

If ~Xml:LoadFromFile('testing.xml',False)

 

 !Find and position to the first parent node

 ! ------------------------------------------

 If ~xml:FindNextNode('Customers','Customer')

         LOOP

             CLEAR(customerqueue);FREE(customerqueue)

             CLEAR(invoicequeue);FREE(invoicequeue)

 

                 XML:AutoRootSet() ! come back to top of customer

 

                 CRecs = XML:LoadQueue(CustomerQueue,True)

 

                 ! Load all Children

                 ! -----------------

             If ~xml:FindNextNode('Invoices','Invoice')

                         IRecs = XML:LoadQueue(InvoiceQueue,False,True)

             End

 

             ! Show what was loaded

             ! --------------------

             XML:DebugMyQueue(CustomerQueue,'Customer Queue')

             XML:DebugMyQueue(InvoiceQueue,'Invoice Queue')

         

                 Do SomeSortOfProcessing

 

                 If XML:GotoSibling() then break.

         End

 End

 Xml:Free()

End
