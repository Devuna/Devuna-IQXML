XML:LoadFromFile

Parameters:

STRING

XML Document File Name (unlimited size file)

<BYTE>

True to Append with existing file in Memory

<BYTE>

True to make all errors silent (No Window)

<LONG>

Override File Access Mode when opening file.

<BYTE>

True to only load a Soap Header




Return Type:

SHORT

XML Error Code (see end of document)

 

Loads an XML Document or JSON Document from a valid ASCII file for processing.

 

If no encoding type is defined in the header,  "US-ASCII" is assumed.

 

Load Soap Header Only: Use True to only load the Soap Header portion of the document if you want to only load the <Soap:Header> to check security or forward the document on to another system or process.  The rest of the document is ignored so this would considerably speed up the load function by ignoring the body portion of the document.

 

Override File Access - By default, the file is opened in Exclusive/Open mode to ensure that the file is not in-use by another process and that the process creating the file has completed.  If you need to override the access to a "SHARE" mode or any other mode, enter a Clarion File Driver "User Access" or "Others Access" mode.

 

The actual values for each access level are:

 

User Access:    

 

 0       0h     Read Only

 1       1h     Write Only

 2       2h     Read/Write

 

Others Access:  

 

 0       0h     Any Access (FCB compatibility mode)

 16      10h    Deny All

 32      20h    Deny Write

 48      30h    Deny Read

 64      40h    Deny None

 

 

See also: XML:SetProgressWindow, XML:AddReplaceEntry, XML:Free
