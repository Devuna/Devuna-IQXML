XML:CloseParent

Parameters:

<STRING>

Optional Programmer Integrity Check flag.  Enter the value of the tag you are expecting to close. If the value does not match the actual tag being close, a programmer error window will appear.
 
These messages will not appear if XML:SkipDebug() flag is set to True even if they cause an error.
 



Return Type:

Does not return a value

 

 

Closes the last parent tag in the stack.

 

Example:

 

XML:CreateParent('MyRootTag');XML:AddParent()

XML:CreateParent('MyRootTag2');XML:AddParent()

XML:CloseParent()

XML:CloseParent()

 

 

<MyRootTag>

<MyRootTag2>

</MyRootTag2>

</MyRootTag>

 

The following would cause a Programmer Integrity Message to appear indicating an error occurred.

 

XML:CreateParent('MyRootTag');XML:AddParent()

XML:CreateParent('MyRootTag2');XML:AddParent()

XML:CloseParent('MyRootTag')
