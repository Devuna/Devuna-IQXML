XML:GetLastString

Parameters:

(None)



Return Type:

STRING

Last Written XML String to Opened File  (limited to 64K)

 

Returns the last written string that was sent to the open file on the thread.  Note that this function will not return any value when using the XML::CreateXMLString() function, and is only valid when using a file.  

 

Note: This function will always return blank unless the "Allow XML:GetLastString" parameter is set to true on the XML:CreateXMLFile() function.

 

Data is cleared, set and can be returned only on the following functions:

 

 XML:AddComment

 XML:AddElement

 XML:AddFromGroup        

 XML:AddFromQueue

 XML:AddParent

 XML:AddStringText

 XML:CloseXMLFile

 XML:CreateXMLFile

 

 

Example:

 

IF NOT XML:CreateXMLFile('sample.xml','1.0','UTF-8',True,,True)

stop(XML:GetLastString)

XML:CreateParent('Root')

XML:AddParent()

stop(XML:GetLastString)

XML:CloseParent()

stop(XML:GetLastString)

XML:CloseXMLFile()

END
