XML:LoadFromString

Parameters:

*CSTRING

XML Document in text (Unlimited String Max)

<BYTE>

True to Append with existing file in Memory

<BYTE>

True to make all errors silent (No Window)

<BYTE>

True to only load a Soap Header




Return Type:

SHORT

XML Error Code (see end of document)



 

Loads an XML Document or JSON Document from passed string text for processing.

 

If no encoding type is defined in the header, "US-ASCII" is assumed.

 

Load Soap Header Only: Use True to only load the Soap Header portion of the document if you want to only load the <Soap:Header> to check security or forward the document on to another system or process.  The rest of the document is ignored so this would considerably speed up the load function by ignoring the body portion of the document.

 

See also: XML:SetProgressWindow, XML:AddReplaceEntry, XML:Free
