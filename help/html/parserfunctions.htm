Here is an overview of what functions to use to complete some common tasks:

 

 

Simple Conceptual Example:

 

 

1. Loading the file

XML:LoadFromFile or XML:LoadFromString

2. Sending Data to Queue

XML:LoadQueue

3. Cleanup

XML:Free


 

Manual Conceptual Example:

 

 

1. Loading the file

XML:LoadFromFile or XML:LoadFromString

2. Finding Data Start Point

XML:FindNextNode

3. Sending Data to Queue

XML:LoadQueue

4. Cleanup

XML:Free

 

Manual Conceptual Example with sub-node processing of three different sets of information:

 

 

1. Loading the file

XML:LoadFromFile or XML:LoadFromString

2. Finding Data Start Point

XML:FindNextNode

3. Setting Root Pivot

XML:AutoRootSet


Finding Child Node <a>

XML:FindNextNode

Sending Data to Queue

XML:LoadQueue(Queue1)

Finding Child Node <b>

XML:FindNextNode

Sending Data to Queue2

XML:LoadQueue(Queue2)

Finding Child Node  <c>

XML:FindNextNode

Sending Data to Queue3

4. Cleanup
