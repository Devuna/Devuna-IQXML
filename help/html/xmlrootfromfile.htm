XML:RootFromFile

Parameters:

STRING

DOS / XML File Name

<LONG>

Override File Access Mode when opening file

*CSTRING

Return Rootname  - Use CSTRING(129)

 

Return Type:

SHORT

XML Error Code (see end of document)

 

Used to return the Root Tag (Beginning/First Tag) in the document.

 

 

Example:                

 

RootName        cstring(129)

 

CODE
 

 If Not xml:RootFromFile('c:\myfile.xml',,RootName)    stop('The first node is ' & RootName)

 end

 

 

Example would be:

 

RootName        cstring(129)

 

CODE
 

 

 If Not xml:RootFromFile('c:\myfile.xml',,RootName) 

    if not xml:LoadFromFile('c:\myfile.xml')

         case RootName

         of 'Customers'

            RunCustomerProcess()

         of 'Invoices'

            RunInvoicesProcess()

         else

                 message('Unknown Document')

         end

     end
 

 end
