SA:ReadFromFile

Parameters:

STRING

File Name (including Path if needed)






Return Type:

STRING

Contents of the file - Up to 4MB

 

Read up to 4MB of text to a file.  When done, check Clarion ERRORCODE() if needed.

 

 

TheResultString                CSTRING(32000)

 

 

 TheResultString = SA:ReadFromFile('c:\debug.log')
