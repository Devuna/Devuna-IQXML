XML:AddFromGroup

Parameters:

GROUP

Local Group to write out to XML file

STRING

Based tag for each record

[<BYTE>

Write out as Attribute Based

<BYTE>

Auto-Close Group Tag (Attribute Based Only)

<BYTE>]

True = Ignore any attributes which have a blank/empty value



Return Type:

Does not return a value

 

Reads a passed GROUP of fields and writes out to an XML document all fields.  Field prefixes are ignored and stripped in field names. This function supports both Groups inside of Groups (for child entries) and Dimmed Fields to write out an XML document using multiple sibling and child records with different parent tags.

 

Fields are written in all uppercase, as that is how the variable is stored in memory in Clarion.  However, if you use a named attribute, then you are able to control the case (or the actual tag name) being written.  This also allows you to have one name in the GROUP, but use a different name that is actually used as the tag name.

 

Ignore any attributes which have a blank/empty value:

Some schemas do not allow blank attributes and will return an error.  You can please True in the 5th parameters to not write out attributes of a blank value.  This would only be valid if the "Write out as Attribute Based" is set to true as well.

 

 

If DIM fields are used, the same amount of dimensions must exist within the group, otherwise, only the first value will be written out.

 

 

NOTE: DIMMED Field and Multi-Group Processing is not valid with Clarion 5.5 Version of iQ-XML

 XML:AddFromGroup(MyGroup,'Customers',True)

 

 

This function is a efficient way of creating an XML document without having to code manually using the XML:AddParent, XML:AddElement functions.  It also allows you to pick fields from a record structure, and write out to a XML document with very little coding.

IF NOT XML:CreateXMLFile('MyFile.XML')

CODE

 

IF NOT XML:CreateXMLFile('Work.XML')

 XML:CreateParent('Division')

 XML:CreateAttribute('Team','Redsox')

 XML:CreateAttribute('Year','1979')

 XML:AddParent()

  CheckOpen(Baseball,1)

  ! Read each record in the file

  SET(TEA:LastNameKey,TEA:LastNameKey)

  LOOP

                 NEXT(Baseball)

               IF ERRORCODE() THEN BREAK.

         ! Pick out only the fields you want to add to the XML Document

         ! ------------------------------------------------------------

         XMLGroup :=: TEA:RECORD

 

         TradeDate = FORMAT(TradeDate,@d010-)        ! fix date

         ! Write out to XML Document with main group being attribute based

         ! ---------------------------------------------------------------

         XML:AddFromGroup(XMLGroup,'Player',True)

 END

  XML:CloseParent()

  XML:CloseXMLFile()

  CLOSE(Baseball)

END
