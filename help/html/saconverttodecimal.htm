SA:ConvertToDecimal

Parameters:

STRING

String to Use



Return Type:

STRING(1K)

Returns a result string

 

 

Looks at a formatted numeric string and converts it into a Numeric or Decimal Number.   Also takes into consideration the negative number in any location (beginning or end) as well as parenthesis.

 

 

 DecNumber        DECIMAL(13,2)

 

 CODE

 

 DecNumber = SA:ConvertToDecimal('1234.23')    

 DecNumber = SA:ConvertToDecimal('(1234.23)')

 DecNumber = SA:ConvertToDecimal('-1234.23')

 DecNumber = SA:ConvertToDecimal('1234.23-')

 DecNumber = SA:ConvertToDecimal('11,234.23')

 DecNumber = SA:ConvertToDecimal('-100,100.23') DecNumber = SA:ConvertToDecimal('(100,100.23)')<

 DecNumber = SA:ConvertToDecimal('$100,100.23') DecNumber = SA:ConvertToDecimal('100,100.23 USD')<
