XML:CreateAttribute

Parameters:

STRING

Attribute Name

STRING

Free-Form Value or Valid IQXML Equate

 



Return Type:

Does not return a value

 

Create an attribute and value to the last created parent tag. You must create a parent before creating an attribute. You can call the XML:CreateAttribute function as many times as needed.   Once you call the XML:AddParent function, the parent and all attributes are written out.   Improper values are corrected for you such as the use of &, where it will be converted to &amp; for you.

 

First, call XML:CreateParent.  Then, call XML:CreateAttribute unlimited times.  Finally, call XML:AddParent.

 

If the second parameter is a valid IQXML Equate (and you are using the IQXML Global Template Extension), the value will be applied.

 

Current supported Equates:

 

IQXML:ISO8601GMT

W3C Date/Time/GMT stamp (yyyy-mm-ddThh:mm:ss-gg:gg) where g is your Windows current GMT adjustment

 

 

 

See also XML:CreateParent, XML:AddParent, XML:SetIgnoreEmpty

 

 

Example:

 

XML:CreateParent('MyRootTag')

 

XML:CreateAttribute('xmlns','http://innquest.com/ns')

XML:CreateAttribute('xmlns:html','http://www.w3.org/1999/xhtml')

XML:CreateAttribute('TimeStamp',IQXML:ISO8601GMT) ! 2005-04-29T10:20:30-05:00
