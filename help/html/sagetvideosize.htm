SA:GetVideoSize

Parameters:

BYTE

1 = Horizontal Size  2 = Vertical Size   3 = Font Adjustment



Return Type:

LONG

 

Returns the current size in Pixels (1 & 2) or Percent (3).

 

 

 

 xml:GetVideoSize(1)        ! returns 640

 xml:GetVideoSize(2)        ! returns 480

 xml:GetVideoSize(3)        ! returns 100
