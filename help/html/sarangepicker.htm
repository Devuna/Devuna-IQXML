SA:RangePicker

Parameters:

*LONG

Return Lower Date

*LONG

Return Higher Date

BYTE

Only allow picking Past Dates and no Future Dates



Return Type:

NONE

 

Puts up a Popup Menu to easily pick a date range. (Today, Last Week, Last Month, Last Year, This Month, This Year, Quarter, etc.)   Really powerful to place on a Report Prompt screen what has a Lower and Higher Date.   There is also a Code Template to simply drop this on a window and handle it for you.

 

 

SA:RangePicker(glo:LowerDate, glo:HigherDate,False)
