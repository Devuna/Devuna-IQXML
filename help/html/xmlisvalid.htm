XML:IsValid

Parameters:

No parameters


Return Type:

Returns 1 (True) if a valid XML Document parsed successfully

 

Returns 1 (True) if the document seems to be a normal XML document that is not missing Tags or Elements.  There may be errors it will not catch (maybe!), but for the most part, if the document is not parsed properly, you can use this to check. &nb

 

If NOT xml:LoadFromFile('myfile.xml',False,True)

 

 if xml:IsValid() <> True

 ! tell the user or write to a log this was not parsed properly

 else

 ! Process the data

 end

 xml:Free()        ! Still needs to be called

 

end
