Template Support

Be sure the iQ-XML template is registered in Clarion after you have installed this program.  You can do this by selecting Setup > Template Registry from within Clarion.  The template is installed in the \[Your Clarion Directory]\accessory\template\win folder for you and is named iQXML.TPL.

 

 

Original Template Author Design Recognition

 

The templates functions to all the API's were originally designed and written by Danie de Beer.  Danie spent a great amount of time writing all the template functions to all the API's for the original release.  I am truly appreciative of his efforts.  Without his efforts, I most likely would never have added template support.

 

Using source code Embeds,  you can use any of the API's via Template Support.  In addition, there is an ImportFromXML and ExportToXML template which will do most (if not all) the work for you.
