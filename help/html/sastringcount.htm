SA:StringCount

Parameters:

STRING

String to Look At

STRING

String to Search For

BYTE

Response Type (0=Count Occurrences  1=Return True if ODD amount  2=Return True if EVEN amount)



Return Type:

SHORT

Response

 

Looks at a string and counts occurrences.

 

Response Type:

 

·If 0, returns the number of occurrences
·If 1, returns True if odd occurrences, False if not odd occurrences and -1 if no occurrences.
·If 2, returns True if even occurrences, False if not even occurrences and -1 if no occurrences.

 

 

Example:

 

 

RCount                SHORT

FullString        CSTRING(512)

 

 

 CODE

 

 FullString = 'Could you tell me this and tell me that?'

 

 ! Returns 2

 Rcount = SA:StringCount(FullString,'tell',0)

 

 ! Returns 1 (True)

 Rcount = SA:StringCount(FullString,'tell',2)
