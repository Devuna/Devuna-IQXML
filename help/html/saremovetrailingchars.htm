SA:RemoveTrailingChars

Parameters:

STRING

A string text

STRING

Character or Characters to remove

<STRING>

Character to not go past when first occurance is found (1 character only)



Return Type:

STRING(10K)

Return string with leading zeros removed

 

Looks at a string, clips and left justifies it and then removes any trailing characters.

 

 

See also SA:RemoveLeadingChars

 

 

Example:

 

 

FullString        CSTRING(32)

 

 

 CODE

 

 ! returns 1234.34

 FullString = '1234.34000000'

 ReturnString = SA:RemoveTrailingChars(FullString,'0')

 

 ! returns 1212

 FullString = 12120.00000

 ReturnString = SA:RemoveTrailingChars(FullString,'0.')

 

 ! returns 12120

 FullString = 12120.00000

 ReturnString = SA:RemoveTrailingChars(FullString,'0.','.')
