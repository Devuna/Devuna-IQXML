Functions

This application is made up of two types of functions (Parser and Writer)

 

Parser Functions:

 

XML:AttributeFieldSet

Sets a field as Required/Optional and/or sets a field picture.

XML:AutoRootClear

Clears the auto position setting of the node pointer.

XML:AutoRootSet

Sets to auto position the node pointer back to a 'root' level after loading Queue

XML:DebugMyQueue

Show the contents of any Clarion Queue for debug purposes.

XML:DebugQueue

Show the XML Queue contents for debug purposes.

XML:FindNextContent

Positions the XML Pointer the closest data content match.

XML:FindNextNode

Positions the XML pointer to the closest specified parent.

XML:Free

Clear the loaded XML Document.

XML:FreeState

Releases a previously stored Saved State.

XML:GetPointer

Retrieves the current record pointer used by iQ-XML.

XML:GotoChild

Positions the XML pointer to the next node deeper within the current node.

XML:GotoParent

Positions the XML pointer to the node previous to the current node.

XML:GotoSibling

Positions the XML pointer to the next node similar to the current node.

XML:GotoTop

Positions the XML pointer to the top/root of the XML document.

XML:LoadFromFile

Load an XML document from passed file name.

XML:LoadFromString

Load a XML document from passed text.

XML:LoadQueue

Loads a local queue with all matched column values with the XML document. (By current node or all for all siblings)

XML:PrimaryFieldCascade

Sets a default value for queue field(s) prior to LoadQueue() function for all new entries created in the queue.

XML:PrimaryFieldClear

Clears primary key fields set in XML:PrimaryFieldCascade() function.

XML:PrimaryKeyCascade

Sets fields to be used to cascade down to other queue records.

XML:PrimaryKeyClear

Clears primary key fields set in XML:PrimaryKeyCascade() function.

XML:ReadCurrentRecord

Returns the contents of the current record.

XML:ReadNextRecord

Advances the current record pointer and returns contents.

XML:ReadPreviousRecord

Set the current record pointer to the previous record and returns contents.

XML:RestoreState

Restores current Node Pointer, current record and data record values.

XML:SaveState

Stores current Node Pointer, current record and data values.

XML:SetPointer

Sets the current record pointer to a specific RRN.

XML:SkipDebug

Skip debug windows even when called.

XML:ViewFile

Opens the XML file (or any file) through Windows registered file type.

 

 

Writer Functions:

 

XML:AddComment

Writes a comment out to an XML file/string.

XML:AddElement

Writes an element out to the XML file/string.

XML:AddFromGroup

Writes out elements for a group structure.

XML:AddFromQueue

Writes out elements for all queue records.

XML:AddParent

Writes the Parent out to the XML file.

XML:CloseParent

Closes the last parent tag.

XML:CloseXMLFile

Closes the XML file and clears all memory used.

XML:CreateAttribute

Prepares the attributes for the parent tag to be written.

XML:CreateParent

Prepares a tag ready to be written.

XML:CreateXMLFile

Creates the XML file for processing.

XML:CreateXMLString

Creates an XML internal string structure for processing.

XML:CloseXMLString

Returns the XML internal string structure.

XML:AddStringText

Writes out the string text to the XML file/string.

XML:SetElementPrefix

Instructs the writer to prefix all elements with prefix/namespace.

XML:SetIgnoreEmpty

Instructs the writer to ignore empty attributes and elements automatically.

XML:CreateEchoToken

Returns a serialized Echo Token to be used for document processing.
