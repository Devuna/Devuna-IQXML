XML:SetNoIndent

Parameters:

BYTE

True or False






 

Tells the Writer not to Indent code (readable form).  Usually used to be more efficient when sending large documents.

 

 

Example:

 

 

      IF NOT XML:CreateXMLFile(WorkFileName,'1.0','')

 

XML:SetNoIndent(True)
