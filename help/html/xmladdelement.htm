XML:AddElement

Parameters:

STRING

Element Name

STRING

Element Value

<BYTE>

Write Value out CDATA.



Return Type:

Does not return a value

 

Writes out an <ElementTag>A Value</ElementTag> to the XML file.

 

See also: XML:SetIgnoreEmpty

 

Example:

 

XML:AddElement('Name','John Doe')

 

<Name>Joe Doe</Name>

 

If the second parameter is a valid IQXML Equate (and you are using the IQXML Global Template Extension), the value will be applied.

 

Current supported Equates:

 

IQXML:ISO8601GMT

W3C Date/Time/GMT stamp (yyyy-mm-ddThh:mm:ss-gg:gg) where g is your Windows current GMT adjustment

 

 

XML:AddElement('TimeStamp',IQXML:ISO8601GMT)
