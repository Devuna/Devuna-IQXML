SA:FormatSQLDate

Parameters:

STRING

An ISO Date or an ISO Date/Time field

<BYTE>

1=Add Day of Week  2=Add Day of Week abbreviated




Return Type:

STRING

Returns a Date formatted as Windows Workstation (@d017)

 

 

Takes an ISO or SQL date (YYYY-MM-DD) and reformats it as a date that is set to be the Windows Workstation Long-Date.  (ie: formats an SQL date better for end-user)

 

The inbound string can be a DATE or a DATETIME string.

 

 

    ! returns  01/01/2016 

MyString = 'The better to read date is ' & SA:FormatSQLDate(SQL:Field1) & '.  Lo

 

    ! returns  01/01/2016 Monday

MyString = 'The better to read date is ' & SA:FormatSQLDate(SQL:Field1,1) & '.  

 

    ! returns  01/01/2016 Mon

MyString = 'The better to read date is ' & SA:FormatSQLDate(SQL:Field1,2) & '.
