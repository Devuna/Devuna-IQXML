XML:Extract

Parameters:

STRING

Left Search Value

STRING

Right Search Value (If not supplied, only Left Search used and will be searched)

STRING

String to Search

*CSTRING

The End Result of what was found

<*CSTRING>

What was to the left of the search string - optional if you don't need it

<*CSTRING>

What was to the right of the search string - optional if you don't need it

LONG

Starting Position (MUST be a 1 or greater)

BYTE

Clean Result --- Set to True to Return End Result without including left/right indicators



Return Type:

LONG

0  = Not Found

-1 = Invalid Data or Search Parameter

any positive number is the location the data was found

 

Extracts a string from another string using a left and right search string.

 

The string text to parse is limited to 4MB.

 

 

Example:

 

 

StrLeft              CSTRING(21)

StrRight             CSTRING(21)

FullString           CSTRING(100)

ExtractedValue       CSTRING(100)

ExtractLeftSide      CSTRING(100)

ExtractRightSide     CSTRING(100)

ExtractedPosition    SHORT

 

 CODE

 

 FullString='This is the full string with \\My Full Variable\\a variable in the middle of it.'

      StrLeft='\\'

      StrRight='\\'

       

      if XML:Extract(Strleft,StrRight,FullString,ExtractedValue,ExtractLeftSide,ExtractRightSide,1,True) = 0

         message('Warning - The Extract Function Failed!!','Failed',icon:exclamation)

      else

         message('Your variable is ' & ExtractedValue & '||New Sentence: ' & ExtractLeftSide & ExtractRightSide,'End Result' )

      end

 

 ! extracted leftside is "This is the full string with "

 ! extracted rightside is "a variable in the middle of it."

 ! extracted Value is "My Full Variable"

 

 

 

The "InBetween" flag will indicate which to include or not include the left/right strings.        

 

 if XML:Extract('*','*','This is *The* value',ExtractedValue,ExtractLeftSide,ExtractRightSide,1,True) = 0

 

! This returns The

 

 if XML:Extract('*','*','This is *The* value',ExtractedValue,ExtractLeftSide,ExtractRightSide,1,False) = 0

 

! This returns *The*

 

 

Example on only using Left Side Search (Simply Break Apart a string such as John|Mary)

 

 if XML:Extract('*','','Hello*There',ExtractedValue,ExtractLeftSide,ExtractRightSide,1,False) = 0

 

 ! ExtractLeftSide = 'Hello'        

 ! ExtractRightSide = 'There'

 ! ExtractValue = '*'
