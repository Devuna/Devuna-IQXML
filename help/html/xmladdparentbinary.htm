XML:AddParentBinary

Parameters:

BYTE

True to auto-close this tag.

STRING

Text or Binary object to encode and attach to XML file

<BYTE>

True to add Microsoft-com and dt:dt description as attributes



Return Type:

Does not return a value

 

Writes out Binary Parent and Attributes.  If the function is called with the "True" parameter,  the parent will be closed with /> instead of >.

 

See also XML:CreateParent

 

Example:

 

XML:CreateParent('MyRootTag');XML:AddParent()

 

XML:CreateParent('Customer')

XML:CreateAttribute(Number,'1111-2222')

XML:AddParent()

 

XML:CreateParent('Signature')

XML:AddParentBinary(True,'signature.gif',True)

 

XML:CloseParent()

XML:CloseParent()
