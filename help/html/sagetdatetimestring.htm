SA:GetDateTimeString

Parameters:

LONG

A Date (zero for current Date)

LONG

A Time (zero for current Time) - Uses @t7
-1 = Do not show any time portion

STRING

Text for Time

BYTE

True to prefix with Day of week

<STRING>

Override Date Picture  (Default to @d4)

<STRING>

Override Time Picture (Default to @t07)



Return Type:

STRING

Returns a Date and Time in a String

 

 

Returns the Date and Time in a string.

 

 

 !   December 21, 2015  Time: 09:12 PM

DateTimeString = SA:GetDateTimeString(TODAY(),CLOCK(),'Time:',false)

 

!   Wednesday, December 21, 2015 at 09:12 PM        

DateTimeString = SA:GetDateTimeString(TODAY(),CLOCK(),'at',True)                

 

 

!   Wednesday, December 21, 2015        

DateTimeString = SA:GetDateTimeString(TODAY(),-1,'',True)
