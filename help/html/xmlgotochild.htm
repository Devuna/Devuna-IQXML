XML:GotoChild

Parameters:

No parameters



Return Type:

SHORT

XML Error Code (see end of document)

 

Advances the cursor location to the child node.  If the child is not found, the cursor location is not changed and an error is returned indicating no children.

 

Attributes and fields containing just values are not considered "child" records.

 

Example 1:

If ~xml:FindNextNode('Customers')

! positions to line 2

If ~xml:GotoChild()

! positions to line 3


If ~xml:GotoChild()

! positions to line 4

If ~xml:GotoChild()

! returns error
