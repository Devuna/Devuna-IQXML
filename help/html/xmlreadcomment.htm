XML:ReadComment

Parameters:

BYTE

Get first Comment or Next Entry (True = Top entry, False = Next entry)

*CSTRING

Comment



Return Type:

BYTE

Error - Returns True if no more comments

 

Used to return the XML comments in the loaded XML file/string.   Comments are limited to 10K each. (10240)

 

Comments are placed in a queue during the XML:LoadFromFile() or XML:LoadFromString() and can be read at any time, until XML:FREE is called.

 

 

Example of reading all comments:

 

In this example, the code will loop through all comments and place them in a local queue.

 

 

StartTop             BYTE

CommentQ             QUEUE,PRE(CQ)

CommentString        CSTRING(2048)

                    END

 

 IF NOT XML:LoadFromFile('soapexample.xml')  

      StartTop = True

      LOOP UNTIL XML:ReadComment(StartTop,CQ:CommentString)

         ADD(CommentQ)

         StartTop = False

      end

   END
