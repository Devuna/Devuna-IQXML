XML:CreateEchoToken

Parameters:

(None)



Return Type:

Returns an Random Echo Token

 

Returns a random serialized Echo Token to be used when processing XML documents such as in SOAP Header.

 

Example:

 

XML:AddElement('EchoToken',XML:CreateEchoToken())

 

<EchoToken>582801a7-c74c-49a5-b0c2-e849817a18556</EchoToken>
