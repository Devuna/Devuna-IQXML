SA:ExtractInside

Parameters:

STRING

String to Search

STRING

Left Side Search

STRING

Right Side Search

<STRING>

L=Keep/Include Left Search Portion   R=Keep/Include Right Search Portion   B=Keep Both Search Strings too

<BYTE>

True to start Right side search from End of String rather than from Next Occurance



Return Type:

STRING(10K)

Returns the inside portion if found

 

Looks at a string and returns portion inside the Left Search and the Right Search.

 

Example:

 

 

 FullString = '<028>001ThisIsRecordOne<028>002ThisIsRecordTwo<028>003ThisIsRecordThree'

 

 ! returns ThisIsRecordTwo

 Result = SA:ExtractInside(FullString,'<028>002','<028>')
