XML:AddStringText

Parameters:

STRING

Any string, including XML Data to be added to the current file or string.

<BYTE>

1 = Escape and convert entire string to escaped XML data before writing out the value.

 

2 = Write As-Is and do not append a CR/LF or any Line Terminator.

 

 

This should usually be False, otherwise < would be converted to &lt;



Return Type:

Does not return a value

 

Adds a string to the current XML document being processed.

 

 

Limitations:

 

·If writing to file (XML:CreateXMLFile) and the 2nd parameter is False, the maximum string length is 65518
·If writing to a file (XML:CreateXMLFile) and the 2nd parameter is True, the maximum string length is 10240
·If writing to string (XML:CreateXMLString) and the 2nd parameter is False, the maximum string length is 4MB
 

Example:

 

XMLEnvelope        string(65518)

 

! assume XMLEnvelope contains proper information

 

IF NOT XML:CreateXMLFile('MyFile.xml')

 XML:AddStringText(XMLEnvelope)

 

 XML:CreateParent('MyRootTag')

 XML:CreateParent('CustomerTag')

 XML:CreateAttribute('Lastname','Smith')

 XML:AddParent(1)

 

 ! Write out <Warning Type="Severe">This can be good &amp; bad error.</Warning>

 XML:CreateParent('Warning')

 XML:CreateAttribute('Type','Severe');XML:AddParent()

 XML:AddStringText('This can be a good & bad error.',True)

 XML:CloseParent()

 

 XML:CloseParent()

 

 XML:CloseXMLFile()

END
