SA:GetFileInfo

Parameters:

STRING

File Name (Full Path if needed)

BYTE

Type of information to send back



Return Type:

STRING

Return result string

 

Returns information about a file

 

Type of Information

 

1.        Long File Name
2.        Date of File
3.        Time of File
4.        Size of File
5.        Attributes of File (See Clarion DIRECTORY)
 
6.        Version of the file if it has EXE/DLL Version information (ie:  Build Version Information)
7.        Return just the file name portion
8.        Return just the path name portion
9.        Return just the extension portion

 

 

TheEXEVersion        STRING(64)

 

! returns 10.0.0.11897

TheEXEVersion = SA:GetFileInfo('c:\clarion10\bin\clarion.exe',6)

 

 

! returns 9.1MB

TheFileSize = SA:FormatFileSize(SA:GetFileInfo('c:\clarion10\bin\clarion.exe',4))

 

 

! returns pdf

TheFileExt = SA:GetFileInfo('c:\clarion10\docs\mydocument.pdf',9)
