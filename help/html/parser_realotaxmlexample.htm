Parser - Real OTA/XML Example

Below is a real extraction from an OTA XML document.  The document is an Open Travel Alliance document sent from Expedia to a hotel to register a hotel booking.  In this example, the data is extracted for multiple room stays, using a SYSID to identify each different room stay.

D1QUEUE QUEUE,PRE(D1)

SYSID

LONG

TimeSpanStart

STRING(10)

TimeSpanEnd

STRING(10)

MarketCode

STRING(4)

SourceOfBusiness

STRING(4)

SpecialRequestName

STRING(50),DIM(5)

END

D2QUEUE QUEUE,PRE(D2)

SYSID

LONG

IsPerRoom

STRING(5)

AgeQualifyingCode

STRING(2)

Count

STRING(20)

END

D3Queue QUEUE,PRE(D3)

SYSID

LONG

RoomTypeCode

STRING(20)

NumberOfUnits

STRING(20)

END

D4Queue QUEUE,PRE(D4)

SYSID

LONG

EffectiveDate

STRING(10)

ExpireDate

STRING(10)

RatePlanCode

STRING(10)

BaseAmountBeforeTax

STRING(20)

BaseCurrencyCode

STRING(5)

BaseDecimalPlaces

STRING(5)

TotalAmountBeforeTax

STRING(20)

TotalCurrencyCode

STRING(20)

TotalDecimalPlaces

STRING(20)

END

 

IF NOT XML:FindNextNode('HotelReservation','RoomStays','RoomStay')

           

StayNDX = 1

LOOP

XML:PrimaryFieldCascade('SysID',StayNdx)

XML:AutoRootSet()

 

XML:QualifyFieldSet('Start')

XML:QualifyFieldSet('End')

XML:QualifyFieldSet('Name')

NDX = XML:LoadQueue(D1QUEUE,True,False)

XML:QualifyFieldClear()

 

IF ~xml:FindNextNode('GuestCounts','GuestCount')

 NDX = XML:LoadQueue(D2Queue,False,True)

END

IF ~xml:FindNextNode('RoomTypes','RoomType')

 NDX = XML:LoadQueue(D3Queue,True,True)

END

IF ~xml:FindNextNode('RoomRates','RoomRate')

 XML:QualifyFieldSet('AmountBeforeTax')

 XML:QualifyFieldSet('CurrencyCode')

 XML:QualifyFieldSet('DecimalPlaces')

 NDX = XML:LoadQueue(D4Queue,True,True)

 XML:QualifyFieldClear()

END

 

XML:PrimaryFieldClear()

 

IF xml:GotoSibling() then break.

StayNDX+=1

END
