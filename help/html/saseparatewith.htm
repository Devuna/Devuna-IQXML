SA:SeparateWith

Parameters:

STRING

Base string

STRING

String to Add

STRING

Character to separate with if base string is not blank



Return Type:

STRING

New Base String

 

Looks to see if it needs to add a separator character.

 

 

 

Example:

 

 

 

 CODE

 

 ! RETURNS NEWOPTION

 BaseString = SA:SeparateWith(BaseString,'NEWOPTION',',')

 

 ! RETURNS NEWOPTION,NEXTOPTION

 BaseString = SA:SeparateWith(BaseString,'NEXTOPTION',',')
