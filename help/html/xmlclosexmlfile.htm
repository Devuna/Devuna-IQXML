XML:CloseXMLFile

Parameters:

<BYTE>

Optional Close Option (0,1 or 2)



Return Type:

Does not return a value

 

 

Closes the current XML document file and flushes the contents to disk.   This also clears any set Field Pictures.

 

 

Close Options

 

·Close Option 1  - Close all unclosed tags that have not been closed before closing the file.
·Close Option 2 - Discard (remove) the XML file.  This is if you change your mind and you want the system to delete what it has created rather than keeping it.

 

 

IF NOT XML:CreateXMLFile('MyXML.XML')

XML:CreateParent('root');XML:AddParent()

XML:AddFromGroup(MyGroup,'Detail',True,True)

XML:CloseParent()        ! root

XML:CloseXMLFile()

END

 

 

IF NOT XML:CreateXMLFile('MyXML.XML')

XML:CreateParent('root');XML:AddParent()

XML:AddFromGroup(MyGroup,'Detail',True,True)

XML:CloseXMLFile(1)         ! close root tag because it was never closed

END

 

 

IF NOT XML:CreateXMLFile('MyXML.XML')

XML:CreateParent('root');XML:AddParent()

XML:AddFromGroup(MyGroup,'Detail',True,True)

XML:CloseXMLFile(2)         ! delete the file / nothing is created

END
