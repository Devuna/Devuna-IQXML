XML:SkipDebug

Parameters:

BYTE

True or False



Return Type:

Does not return a value

 

Setting this value to True will skip all DebugQueue and DebugMyQueue functions called.  Setting this value is Global – Not Threaded.

 

See also XML:DebugMyQueue, XML:DebugQueue

 

 

Why would I want to do this?  

If you want to release your application with debug points, and only execute them when something goes wrong.

 

 

Example:

 

 

 ShowDebug = GETREG(REG_LOCAL_MACHINE,'SOFTWARE\My Company\MyProduct','DebugXML')

 if ShowDebug <> True

         XML:SkipDebug(True)

 end

 

If NOT XML:LoadFromFile(XMLFileName)

 If NOT XML:FindNextNode('Customers','Customer')

         XML:DebugMyQueue(MyQueue,'MyQueue Contents')

         

 

 

In the above example, the Debug screens will always be ignored, unless the user sets a switch in the registry rather than having to recompile the program with debug points every time.
