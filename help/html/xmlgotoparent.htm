XML:GotoParent

Parameters:

No parameters



Return Type:

SHORT

XML Error Code (see end of document)

 

Advances the cursor location to the parent node.  If the parent is not found, the cursor location is not changed and an error is returned indicating top of XML file.

 

If you are have already been on a "parent record", and you wish to return back to it after sub-level processing, XML:SaveState and XML:RestoreState is more efficient than using XML:GotoParent.
