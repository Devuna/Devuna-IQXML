XML:ReadNextRecord

Parameters:

*QUEUE

Queue to contain elements at current record (Declare with one STRING)

*CSTRING

Data value at current location



Return Type:

SHORT

XML Error Code (see end of document)

 

Used to read the next relative record and return the values. The Queue will contain a record for each element and the data value will contain the value.

QUEUE,PRE(QUE)

ElementName

 END

ElementValue
