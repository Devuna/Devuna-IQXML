XML:GetSoapContent

Parameters:

STRING

Entire node (excluding Soap Nodes)

*CSTRING

CSTRING Variable to hold value being returned



Return Type:

BYTE

True if node found / False if not successful

 

Used to quickly retrieve the value of an element within the Soap Header.  The first node must begin with "Soap" in order for this command to retrieve a value.  This allows quick access to a Soap header value without having to use the FindNextNode or LoadQueue functions.  It is very efficient to quickly read the contents of a single node, to make a decision for later parsing.

 

The Node must be in the form of Node[.Node.Node] excluding any SOAP portion.

 

See also: XML:GetRawContent

 

Example 1:        

 

 

<?xml version="1.0"?>

<soap:Envelope xmlns:soap="http://www.w3.org/2001/12/soap-envelope"

soap:encodingStyle="http://www.w3.org/2001/12/soap-encoding">

 

<soap:Body xmlns:m="http://www.example.org/stock">

 <m:GetStockPrice>

   <m:StockName>IBM</m:StockName>

 </m:GetStockPrice>

</soap:Body>

 

</soap:Envelope>        

 

 

 

! Retrieve the StockName to run the routine to get the stock price

 

StockName        CSTRING(32)

 

if not xml:LoadXMLFile('FileAbove.XML')

 

 if xml:GetSoapContent('m:GetStockPrice.m:StockName',StockName)

         

         Do GetStockPrice

 

 end

end

 

 

Example 2:        

 

 

<SOAP-ENV:Envelope  xmlns:SOAP-ENV=http://schemas.xmlsoap.org/soap/envelope/>

   <SOAP-ENV:Header>

        SOAP-ENV:RequestType="CheckRate" />

   </SOAP-ENV:Header>

   <SOAP-ENV:Body>

 <Rates>

 <Rate Type='Test'>The test Rate</Rate>

 </Rates>

   </SOAP-ENV:Body>

</SOAP-ENV:Envelope>

 

 

 

RequestType        CSTRING(32)

 

if not xml:LoadXMLFile('FileAbove.XML')

 

 if xml:GetSoapContent('RequestType',RequestType)

 

         case RequestType

         of 'CheckRate'        

                 Do CheckRateRoutine

         of 'CheckInventory'

                 Do CheckInventoryRoutine

         of 'CheckGuest'

                 Do CheckGuestRoutine

         else

                 Do UnknownRequestRoutine

         end

 

         Do SendResponseBack

 

 end

end
