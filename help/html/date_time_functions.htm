Date/Time Functions

Calendar, Range Picker, Date and Time functions

 

·SA:CalendarPicker - Popup Calendar
 
·SA:FullTimeSinceNow - Show the time since now in a string (including range of days)
 
·SA:GetDateTimeSimple - Show the DATE and TIME in a single 16 character string (DDMMYYHHMMSS)

 

·SA:GetDaysInMonth - Get days in a month
 
·SA:GetDOW - Get day of the week
 
·SA:GetMonthName - Get Month Name
 
·SA:GetSecondsString - Get number of seconds that have transpired in a string (ex:  1 minute and 2 seconds)
 
·SA:RangePicker - Prompt the user for a range of dates nicely through a popup menu (this week, next week, this month, next month, etc)
 
·SA:MinutesSinceNow - Get exact number of minutes since now
 
·SA:TimeSinceNow - Show the time since now in a string (none range of days)
