XML:RootFromString

Parameters:

STRING

String of XML Data

 

Return Type:

STRING

Root Name (Blank if not found)

 

Used to return the Root Tag (Beginning/First Tag) in the document.

 

 

Example:                

 

 

CODE
 

 If HTTPGetData('127.0.0.1',80,'/xml/datarequest.php?user=1234&pw=1234',ReturnData,ReturnError) = NO_ERROR

         if not xml:LoadFromString(ReturnData)

 

                 case xml:RootFromString(ReturnData)

                 of 'Customers'        

                         RunCustomerProcess()

                 of 'Invoices'        

                         RunInvoiceProcess()

                 end

 

         end
