SA:ClipAndJustify

Parameters:

STRING

String to Use

<STRING>

Format Picture - if blank, Static Picture will be used



Return Type:

STRING(1K)

Returns a result string

 

Looks at a string, formats it, and the left justifies it and then clip it.   Used to quickly get a good result for a numeric or date picture, without having to left justify or clip the result.  Also, you can set your Static Picture using SA:ClipAndJustifySet(your picture) and then call this parameter easily, good when using a local currency picture.

 

 

 CODE

 

 ! returns LEFT/CLIPPED string of -123.21

 ResultString = SA:ClipandJustify(-123.21,'@n-12.2')        

 

- or -

 

 SA:ClipAndJustifySet('@n$-13.2')

 

 ! returns LEFT/CLIPPED string of $-123.21

 ResultString = SA:ClipandJustify(-123.21)                

 

 

 message('You have to pay us ' & SA:ClipAndJustify(ThisAmount) & ' or we will send to collections'

 

 

 The SA:ClipandJustifySet( ) picture is set GLOBALLY for the ENTIRE application, and all threads until changed.

 

 

Parameters:

STRING

Static Format Picture (Global/All Threads)



Return Type:

NONE

 

 

  SA:ClipandJustifySet('@n$-13.2')
