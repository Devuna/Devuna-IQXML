XML:GotoSibling

Parameters:

No parameters



Return Type:

SHORT

XML Error Code (see end of document)

 

Advances the cursor location to the next exact like node.  If the sibling is not found, the cursor location is not changed and an error is returned indicating no more siblings. If there is a node attached to the parent, with a different sibling name,  the function will return an error.

 

To find attached siblings of a different name within the XML document, you must use the XML:FindNextNode function.

 

1.<xmlmessage>
2.<Customers>
3. <Customer>
Example 1:

If ~xml:FindNextNode('Customers')

! positions to line 2

If ~xml:GotoSibling()

! returns error

 

Example 2:

If ~xml:FindNextNode('Customer')

! positions to line 3

If ~xml:GotoSibling()

! positions to line 14

 

Example 3:

If ~xml:FindNextNode('Customer','Address')

! positions to line 7

If ~xml:GotoSibling()

! returns error

 

Example 4:

If ~xml:FindNextNode('AddressLine')

! positions to line 8

If ~xml:GotoSibling()

! positions to line 9

If ~xml:GotoSibling()

! returns error
