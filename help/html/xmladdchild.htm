XML:AddChild

Parameters:

STRING

Name of Element



Return Type:

Does not return a value

 

Creates an empty Parent (without Attributes).

 

This is the same as called XML:CreateParent() and XML:AddParent()

 

Example:

 

 XML:AddChild('root')

 XML:AddChild('information')

 xml:CloseParent()

 XML:CloseParent()

 

<root <information </information</root 

 XML:CreateParent('root');XML:AddParent()

 XML:CreateParent('information');XML:AddParent()

 xml:CloseParent()

 XML:CloseParent()

 

 

(to Auto-Close an Element)

 

 XML:AddChild('root')

 XML:CreateParent('information');XML:AddParent(True)

 XML:CloseParent()

 

<root <information/</root XML:AddChild('root')

 XML:CreateParent('information')

 XML:CreateAttribute('type','detail')

 XML:AddParent(True,'This is an XML example')

 XML:CloseParent()
