XML:ReadCurrentRecord

Parameters:

*QUEUE

Queue to contain elements at current record (declare with one STRING)

*CSTRING

Data value at current location



Return Type:

SHORT

XML Error Code (see end of document)

 

Used to read the XML record at the current cursor location. The Queue will contain a record for each element and the data value will contain the value.

 

 

Example of reading current record and display the node name:

 

In this example, the program will display a box with showing the current node pointer and value.

 

<xmlmessage<Customers  <Customer     <PersonName <GivenName 

MyQueue

QUEUE,PRE(QUE)

ElementName

STRING(35)

 END

ElementValue

CSTRING(128)


Recs

SHORT

Node

STRING(2048)

 

If ~xml:ReadCurrentRecord(MyQueue,ElementValue)

 Loop Recs = 1 to records(MyQueue)

         Get(MyQueue,Recs)

         Node = clip(Node) & chr(60) & clip(que:elementname) & chr(62)

 End

 Message('Currently at: ' & clip(Node) & '||Value: ' & clip(ElementValue))

End

 

 

Example of reporting number of levels:

 

If ~xml:ReadCurrentRecord(MyQueue,ElementValue)

 MESSAGE('There are ' & records(MyQueue) & ' elements at current location.')

End

 

 

Example to see what kind of "Transaction" this is:

 

In this example, you may need to know something about the current record you are on, but the element could be located at any level.  This might be because in an XML document, it could be wrapped with other parent tags you are not concerned with or

 

If ~xml:ReadCurrentRecord(MyQueue,ElementValue)

 loop Recs = 1 to records(MyQueue)

         Get(MyQueue,Recs)

         if upper(que:ElementName) = 'OTA_ADDTRANSACTION'

                 TranType = 'Add'

                 break

         elsif upper(que:ElementName) = 'OTA_CHANGETRANSACTION'

                 TranType = 'Change'

                 break

         end

 end
