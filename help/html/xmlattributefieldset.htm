XML:AttributeFieldSet

Parameters:

STRING

Field name in LocalQueue

<BYTE Required>

IQXML:NONE (0)

IQXML:REQUIRED (1)

IQXML:OPTIONAL (2)
IQXML:FORCEWRITE (100)

 

<STRING Picture>

A valid Clarion picture to format the string before being placed in the Queue field.

 



Return Type:

Does not return a value

 

Further defines a field for the XML:LoadQueue() function.  If you have added the global template, you can use the Equates.  These values are not cleared until the XML:AttributeFieldClear() function is called or the XML:Free statement is calle

 

Required:

 

IQXML:REQUIRED

If this field is blank, the record will not be added to the passed Queue, even if other fields contain values.

 

IQXML:OPTIONAL

If this field has a value, a record will not be added to the passed Queue unless another non-optional field is populated.

 

IQXML:FORCEWRITE

A record will be inserted into the queue immediately when this field or attribute is reached, regardless of whether or not all, some or none of the fields have data in them.   Used when you know th  

 

String values in an XML document which contain "0" or "false" or "0.00" will be considered a 'filled' field.  Only BLANK
