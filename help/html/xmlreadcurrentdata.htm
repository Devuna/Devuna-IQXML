XML:ReadCurrentData

Parameters:

*CSTRING

Value of current element



Return Type:

SHORT

XML Error Code (see end of document)

 

Used to read the XML data value at the current cursor location.  The value will remain unchanged if an error occurs.

 

 

Example of reading current data:

 

In this example, the program will display a box showing the current data at the current node pointer.

 

<xmlmessage>

<Customers>

 <Customer>

    <PersonName>

 <GivenName>Robert</GivenName>

 

 

MyAnswer        CSTRING(32)

 

 if NOT XML:FindNextNode('GivenName')

         IF NOT XML:ReadCurrentData(MyAnswer)

                 MESSAGE('The name is: ' & MyAnswer)

         END

 END
