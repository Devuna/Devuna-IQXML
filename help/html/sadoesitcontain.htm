SA:DoesItContain

Parameters:

STRING

String to Search

STRING

Item to Search For

<BYTE>

Case Sensitive Search (True or False)



Return Type:

BYTE

Returns True if exists or False if it does not

 

 

Looks at a string and returns True if it is found.

 

 

 PassiveTransfer = SA:DoesItContain(FullParms,' -PASV') 

 SecureTransfer = SA:DoesItContain(FullParms,' -SECURE') 

 DeleteAfter = SA:DoesItContain(FullParms,' -DELETEAFTER')
