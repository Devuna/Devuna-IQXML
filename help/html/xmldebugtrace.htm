XML:DebugTrace

Parameters:

<BYTE>

True=On   False=Off



Return Type:

Does not return a value

 

Turns on automatic step tracing with the XML:FindNextNode statement.   When this is set to on, all calls to the XML:FindNextNode function will automatically invoke the internal debugger (XML:DebugQueue).

 

 

This may help you see what the XML:FindNextNode is doing, if it is working or if it is failing.  

 

When "On", the XML:FindNextNode will invoke the debugger twice every time automatically.  Once when the function is called, showing you the node pointer before the find is executed.  Then, at the end of the call, it will show you the new node pointer or if an error occurred.
