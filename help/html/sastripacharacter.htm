SA:StripACharacter

Parameters:

STRING

String to look at for stripping (Up to 4MB in length)

STRING(1)

A character to strip (Must be one character)



Return Type:

STRING(10K)

Return result string

 

 

Looks at entire string and strips all occurrences of the character.

 

 

! 1,234.56 will return 1234.56

TheDollarValue = SA:StripACharacter(DollarValue,',')
