XML:AddReplaceEntry

Parameters:

<STRING>   Replace what

<STRING>   Replace with


Return Type:

Does not return a value

 

Unlimited calls this function can be made to setup different string replacements of values when loading an XML file. The replacements must be set prior to calling XML:LoadFromFile and XML:LoadFromString.

 

The system automatically translates encoded, HTML and ASCII characters such as &nbsp;  &apos;  &#020; and anything else that is common.  However, you may have to do some additional translation for languages or special characters you encounter in your XML file.

 

All replace values are cleared when XML:Free() is called.

 

 

Example:        

 

 

 CODE

 

 XML:AddReplaceEntry('&#184;', ' ')

 XML:AddReplaceEntry('Robert','James')

 XML:AddReplaceEntry('<space>',' ')

 

 

 IF NOT xml:LoadFromFile('c:\myxmlfile.xml')

 

         ! your processing

 

 END

 

 XML:FREE()

 

 

See also XML:LoadFromfile, XML:LoadFromString, XML:Free
