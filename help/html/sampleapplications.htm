Sample Applications

This topic will display different examples of Parser and Write applications.  Most of these applications are available in the ExamplesForiQXML.app which was installed in the %_CSIDL_COMMON_DOCUMENTS%\Softvelocity\Clarion10\Accessory\iQXML\Examples folder.

 

The Examples app needs the ASCII and TPS file drivers.

The recent.app needs the ASCII file driver.

 

When using iQ-XML in a end-user app, you just need to include the DOS (CLADOS.DLL) file driver and the iqxml.dll
