Clarion Template Support:

It is recommended to register the iqxml template after installation.  Once you do that, simply add the Global Extension to your Clarion application before using.

 

 

iQxml.tpl

This file will be placed in your Clarion Templates directory during the installation but not automatically registered.  Simply register the iqxml.tpl template from within Clarion.
