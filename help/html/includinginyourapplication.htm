Including in your Application

These iQ-XML functions allow you to easily process standard XML documents within your Clarion application.  All functions are included in one DLL which needs to be attached and distributed with your application in order for it to work.  If you are not already distributing the DOS File Driver, you must do that as well.

 

 

 

FOR CLARION 10.x

 

IQXML.DLL

All XML functions for Clarion 10.x (Installed in the Clarion10\acessory\bin folder)

CLADOS.DLL

Clarion DOS File Driver (for Clarion 10)

 

 

 

Clarion Template Support:

The iqxml template is automatically registered by the installation program.  Simply add the Global Extension to your Clarion application before using.

 

 

iQxml.tpl

This file will be placed in your Clarion Templates directory during the installation and automatically registered.

 

Add the iQ-XML Global Template option to your Clarion or ABC application which will automatically add the Module Includes for you.

 

In Source Code embeds, you can select any of the iQ-XML functions or manually code them yourself.
