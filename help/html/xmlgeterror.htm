XML:GetError

Parameters:

<SHORT>

Error code returned by other XML functions

 

Return Type:

STRING

Error Description

 

Used to return the error text of an error code returned from other iQ-XML functions.

 

Example:                        

 

Error_Code = xml:restoreState(CustomerPointer)

If Error_Code

       ErrorText = XML:GetError(Error_Code)

 Message('Error Occurred!||Error: ' & ErrorText)

End

 

Calling this function without a parameter will automatically return back the last error that has happened on the current thread.  If the last function called ended without error, no error response will be returned.

 

If xml:restoreState(CustomerPointer)

 Do ShowError

 Exit

End

 

ShowError        ROUTINE

 

 Message('An error occurred in XML parser.||' & XML:GetError(),'Error')
