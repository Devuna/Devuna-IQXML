XML:PrimaryFieldClear

Parameters:

No parameters


Return Type:

Does not return a value

 

Clears all the Primed Fields recorded to the buffer set in the XML:PrimaryFieldCascade() function.

 

See also XML:PrimaryFieldCascade

 

 

Example:

 

! Write out all records to the queue with 'Company' field as ABC Company

 

XML:PrimaryFieldCascade('Company','ABC Company')

If NOT XML:FindNextNode('Data')

 AddedRecords = XML:LoadQueue(MyQueue,True,True)

End

XML:PrimaryFieldClear()
