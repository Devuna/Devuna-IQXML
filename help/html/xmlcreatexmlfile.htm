XML:CreateXMLFile

Parameters:

STRING

XML File name

<STRING>

XML Version

<STRING>

XML Encoding

<BYTE False>

Add Version/Date/Time Stamp comment

<STRING>

Line Terminal Characters (maximum of 2)  - The default is CR/LF.

<BYTE false>

Allow XML:GetLastString function to return its value.

<BYTE false>

Allows to Append existing file when True



Return Type:

SHORT

Clarion Error Code

 

Creates an XML file ready for processing.

 

Only one XMLFile or XMLString can be in use at one time.  If a previous XML:CreateXMLFile or XML:CreateXMLString has been called without a close, an errorcode 5 (Access Denied) will be returned.

 

To create a file without any header data being written (ie: <xml), use the following:

 

 IF NOT XML:CreateXMLFile('output.xml',,,'')

 

 

Encoding:

 

·UTF-8 changes all High-ASCII characters (128 or higher except Euro sign) to Double-Byte Characters compatible with XML viewers and parsers.   If you are going to transmit extended characters (Danish, Russian, Croatian, etc.) Attributes and tags are not encoded, data is encoded.
 
·ISO-8859-1 (Default) accepts all ASCII characters (Latin and extended ASCII) - All characters are coded in single eight-bit code and unconverted.

 

 

NOTE: You can also use XML:CreateAttribute function before calling this function to add additional attributes to the root tag.

 

Examples:

 

IF NOT XML:CreateXMLFile('c:\outbound\myfile.xml','1.0','ISO-8859-1',True)

XML:CreateParent('Root');XML:AddParent()

XML:CloseParent()

XML:CloseXMLFile()

END

 

 

<?xml version="1.0" encoding="UTF-8" ?>

<-- Created by iQ-XML Version 1.00   Date: 01/02/2005  Time: 10:59PM -->

 <Root>

</Root>

 

This example uses no Line Terminator character causing all data to be on one line.  Be careful that the receiving parser can handle unlimited field length data.  It is recommended not to use this.   If you are sending to a MAC or MAINFRAME, you may simply want to use a LF character such as CHR(10).

 

IF NOT XML:CreateXMLFile('c:\outbound\myfile.xml','1.0','ISO-8859-1',True,'')

 XML:CreateParent('Root');XML:AddParent()

 XML:CloseParent()

 XML:CloseXMLFile()

END

 

<?xml version="1.0" encoding="UTF-8" ?><-- Created by iQ-XML Version 1.00   Date: 01/02/2005  Time: 10:59PM --><Root></Root>

 

 

You can create an empty "shell" file for writing data to by creating the file without any Version or Encoding, and setting the Line Terminal Character to nothing.

 

 IF NOT XML:CreateXMLFile('c:\myemptyfile.txt',,,'')

         XML:AddStringText('Hello there')

 END

 

 

Here is an example of adding more attributes to the root tag:

 

 XML:CreateAttribute('standalone','yes')
