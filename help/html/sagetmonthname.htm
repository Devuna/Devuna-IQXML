SA:GetMonthName

Parameters:

LONG

Date or Index

<BYTE>

Max Characters to return



Return Type:

STRING

Return Day of Month (English)

 

Returns the name of the Month.

 

If Value is 1-12, returns name of month (January, February, March, April, etc.)

If Value is > 12, it returns the day of the month of the Clarion Date

 

 

! Returns January

TheMonth = SA:GetMonthName(ClarionDate)

 

! Returns Jan

TheMonth = SA:GetMonthName(ClarionDate,3)

 

! Returns J

TheMonth = SA:GetMonthName(ClarionDate,1)

 

! Returns January

TheMonth = SA:GetMonthName(1)
