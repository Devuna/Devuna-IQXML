XML:ReadCurrentNode

Parameters:

None



Return Type:

STRING

Current Node

 

Used to read the current node.  This is where the pointer currently is.   If no node is active, then blank is returned.

 

This is simply returning the value of the current node pointer's highest value.

 

 

<xmlmessage<Customers  <Customer     <PersonName <GivenName 

 

MyAnswer        CSTRING(32)

 

 if NOT XML:FindNextNode('GivenName')

         MESSAGE(XML:ReadCurrentNode)        ! returns givenname  (Note, always lowercase)

 END
