SA:DoesItContainAny

Parameters:

STRING

String to Search

STRING

Items to Search For seperated by pipe |

BYTE

Case Sensitive Search (True or False)



Return Type:

STRING(10K)

Returns the index it found it in or 0 if not found.

 

 

Looks at a string and returns the index where it exists from an Items List.

 

 

 if sa:DoesItContainAny         'Windows XP|Windows 98|Windows 95|Windows ME|Windows 2000|Windows 2003',True) 

         message('This op system is not supported')
