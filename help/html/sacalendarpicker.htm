SA:CalendarPicker

Parameters:

LONG

Starting Date

<STRING>

Window Title

<BYTE>

True (1) to show as a smaller calendar window



Return Type:

LONG

Return date picked

 

Pops up a calendar and returns a date.  The system takes into consideration the starting day (Sunday or Monday) from the Windows Setting.

 

There is also a Code Template to simply drop this on a window and handle it for you.

 

 

TheDatePicked        LONG

 

TheDatePicked = SA:CalendarPicker(TODAY(),'Pick a Date',True)
