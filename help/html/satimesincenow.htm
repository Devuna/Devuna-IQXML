SA:TimeSinceNow

Parameters:

LONG

Start Time


BYTE

Digits of Milliseconds to show (0,1 or 2)



Return Type:

STRING

Return Time elapsed (HH:MM:SS.ss) since now or current CPU time

 

Returns the time elapsed since now.  This does not take into consideration a change in days, so to be used for debug information or when the entire time is less than 24 hours.   To use time since current that would span a day or days or be more than 24 hours, see SA:FullTimeSinceNow

 

 

! 00:00:23.09

TimeElapsed = SA:TimeSinceNow(StartTime,2)                

 

! 00:00:23.1

TimeElapsed = SA:TimeSinceNow(StartTime,1)        

 

! 00:00:23        

TimeElapsed = SA:TimeSinceNow(StartTime,0)        

 

 

You can use this to get the current time with milliseconds by sending 0 as the Start Time.

 

! 12:23:50.45  (good for debug logging)

ExactCPUTime = SA:TimeSinceNow(0,2)
