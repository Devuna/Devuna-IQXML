SA:WriteToFile/SA:WriteToFileA

Parameters:

STRING

File Name (including Path if needed)

STRING

String text to write (up to 4MB)

<BYTE>

True Append file - False (Default) to create a new file

<BYTE>

0 (Default) Write as-is with CR/LF at end

 
1=Add Date/Time stamp (@d01)
2=Add Date/Time/Thread stamp (@d01)

11=Add Date/Time stamp (@d017)
12=Add Date/Time/Thread stamp (@d017)
 

20=Write data using an (Decimal/ASCII) dump for each character

 

30=Write as-is without any CR/LF (Raw dump of data completely as-is)

 






Return Type:

NONE

 

Write up to 4MB of text to a file.  When done, check Clarion ERRORCODE() if needed.

 

Time Stamp:

 

0 = Write text as is
 

1 = Write text but prefix with Date (@d01), Time such as 01/15/14  12:12:09.09  Debug Point

2 = Write text but prefix with Date (@d01), Time and Thread such as 01/15/14  12:12:09.09  002  Debug Point

 

11 = Write text but prefix with Date (@d017), Time such as 01/15/14  12:12:09.09  Debug Point

12 = Write text but prefix with Date (@d017), Time and Thread such as 01/15/14  12:12:09.09  002  Debug Point

 

 SA:WriteToFile(glo:DebugFile,'Write 20&nb

 

 

If you use option 20 (Write data using an ASCII Dump, it would look like this: (16 bytes per line)

 

 048 048 048 048 054 048 048 048 013 010 031 139 008 000 000 000

 000 000 000 000 236 189 217 114 027 073 178 032 250 044 153 205

 063 068 161 170 005 170 011 009 032 177 131 034 089 003 046 146

 

 

SA:WriteFile

 

To make debugger easier, you can use SA:WriteLine once the file and attributes have been set.

 

 ! Set the defaults

 SA:WriteToFileSetName('c:\debug.log',True,2)

 

 SA:WriteLine('This is line 1')

 SA:WriteLine('This is line 2')

 SA:WriteLine('This is line 3')

 SA:WriteLine('This is line 4')

 

SA:WriteToFileA

 

Writes/dumps raw data to disk from a Memory Address (*CSTRING) rather than a passed STRING parameter.  This is faster and more memory efficient if you are dumping a large string to disk.   Make sure it

 

 SA:WriteToFileA('c:\debug.log',CStringVariable,False)
