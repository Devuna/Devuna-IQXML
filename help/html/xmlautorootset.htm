XML:AutoRootSet

Parameters:

No parameters


Return Type:

Does not return a value

 

Instructs the system to mark the current node pointer as the 'pivot' location to always return to after a XML:LoadQueue function.

 

This is usually used when processing structures with multiple XML:FindNextNode and XML:LoadQueue functions. The system will always return the current record pointer and node pointers to the set location after all XML:LoadQueue functions and will do so until it has been cleared with XML:AutoRootClear or has been cleared by XML:Free.

 

You can call XML:AutoRootSet if it is already "in-use", and it will simply just set to the new pointer value.

 

Important Note: Only one XML:AutoRootSet() function can be in-use at one time.  If you need multiple pivot locations, you must use the XML:SaveState and XML:RestoreState functions.

 

See also XML:AutoRootClear

 

Example:

 

! Load a bunch of queue's with the proper information, process it

! and then move on to next customer.  After every XML:LoadQueue

! statement, the cursor/node pointer is moved back to the top of the

! current customer record. This allows all child nodes to appear in any

! order within the parent and still process correctly.

 

If NOT XML:FindNext('Customer')

       LOOP

         XML:AutoRootSet()

         XML:LoadQueue(HeaderQueue,false,false)                        ! name info

 

         If Not XML:FindNext('Address')

                   NDX = XML:LoadQueue(AddressQueue,True,True)        ! addresses

         end

         If Not XML:FindNext('Comments')

                   NDX = XML:LoadQueue(CommentQueue,True,True)        ! comments

         end

         If Not XML:FindNext('Invoices')

                   NDX = XML:LoadQueue(InvoiceQueue,True,True)        ! invoices

         end

         If Not XML:FindNext('Totals')

                   NDX = XML:LoadQueue(TotalsQueue,True,True)        ! totals

         End

 

         Do CustomerProcessingRoutine

 

         FREE(TOTALSQUEUE);FREE(INVOICEQUEUE);FREE(COMMENTS)

         FREE(ADDRESSQUEUE);FREE(HEADERQUEUE)

 

         CLEAR(TOTALSQUEUE);CLEAR(INVOICEQUEUE);CLEAR(COMMENTS)

         CLEAR(ADDRESSQUEUE);CLEAR(HEADERQUEUE)

 

         If XML:GotoSibling() then break.

 END

END
