SA:StripAllButNumbers

Parameters:

STRING

String to look at for stripping

<BYTE False>

True to also allow minus sign (-) and period/decimal (.)



Return Type:

STRING(10K)

Return result string

 

Looks at entire string and strips all non numeric characters.  This leaves only characters 0 through 9.  If the parm is true, then it also allows periods and negative signs.

 

 

! returns 12000.12

TheDollarValue = SA:StripAllButNumbers('$12,000.12',True)

 

! returns -12000.12

TheDollarValue = SA:StripAllButNumbers('$-12,000.12',True)

 

! returns 1200012

TheDollarValue = SA:StripAllButNumbers('$12,000.12',False)
