XML:DebugQueue

Parameters:

<STRING>

Window Title



Return Type:

Does not return a value

 

Shows the current loaded XML document in memory in an iQ-XML Internal Structure format for debugging.  You can then optionally see the current node pointer, Qualified Fields set, Primary Keys set and Attribute Fields set. You can then manually navigate through the XML structure and see where your program's current node pointer is.  You can also highlight the record set (elements) that your document is currently reading.

 

 

See also XML:DebugMyQueue, XML:DebugTrace

 

Example 1:        

 

 if NOT xml:LoadFromFile('Myfile.xml')

           XML:DebugQueue()

 End

 

 

Example 2:

 

 XML:DebugQueue('Before FindNextNode')

 IF NOT XML:FindNextNode('XMLMessage','Systems','System')

         XML:DebugQueue('After FindNextNode')

 

         LOOP

                 NDX = XML:LoadQueue(MyQueue,False,False)

                 IF XML:GotoNextSibling THEN BREAK.

                 XML:DebugQueue('On to next sibling')

         END

 END
