Special Functions

 

·SA:ChangeColorTint - Take a RGB color and make it lighter or darker
 
·SA:GetOPSystem - Get the Version, Edition and Build of the operating system as well as if running under Terminal Services
 
·SA:RGBTODecimal - Take a Clarion String Color (FFDDEE) and convert it to LONG
 
·SA:WriteToSysDebugger - Write string of text to SysInternals Debugger (Real time debugging)
 
·XML:ViewFile - Take a string and open the right application with it - including MailTo, HTTP, PDF's, Text, Logs, etc.  (Will use UltraEdit for a TXT file for example)
