XML:GetPointer

Parameters:

No parameters



Return Type:

LONG

Current Cursor Location

 

Used to retrieve the current cursor location to resume sequential processing.

 

See also XML:SetPointer

 

 

Example:        

 

SavePointer = XML:GetPointer

! do some sort of processing on next record

If ~xml:ReadNextRecord(MyQueue,MyData)

 Do ProcessingRoutine

End

                                 

! Go back to the original record

If XML:SetPointer(SavePointer)

 Message('Error occurred!','Error!')

End
