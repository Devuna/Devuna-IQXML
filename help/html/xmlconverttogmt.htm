XML:ConvertToGMT

Parameters:

<LONG>

Date (blank is today)

<LONG>

Time (blank is now)



Return Type:

STRING

The current time in full ISO GMT format

 

Gives you the current time in full ISO (date, time, zone adjustment)

 

 

 YYYY-MM-DDTHH:MM:SS(zone adjustment from GMT)

 

 2009-06-30T16:10:18-04:00

 

 

Example:

 

 

TheTimeToSend        string(25)

 

 CODE

 

 TheTimeToSend = XML:ConvertToGMT()
