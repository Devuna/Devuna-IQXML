SA:ExtractParm

Parameters:

STRING

String to Search

STRING

String to Search For

BYTE

Case Sensitive Search (True or False)

*BYTE

Return 1 (True) if Parm exists or 0 (False) if it did not



Return Type:

STRING(10K)

Returns the Search String with the Search For string removed

If the search is False (Not found), then the original string is returned in full

 

Looks at a string and returns if the Search value exists as well as returns the search string with the value removed.

 

 

 ! look to see if -PASV exists in the string, and if so, rem

 

         ! g_strHostName = 'www.mywebsite.com\directory -PASV'

 

 OverridePassive        BYTE

 g_strHostName                STRING(384)

 

 g_StrHostName = SA:ExtractParm 

 ! Returns:
 

 !  OverridePassive = 1 (True)    

 !  ReturnString    = 'www.mywebsite.com\directory'
