XML

:Replace

Parameters:

STRING

Original String

STRING

Replace Text (case insensitive)

STRING

Replace With

<LONG> - Multi-Purpose

Starting location for replacement.   Or, use -1 to match "as-is" and not ignore case.



Return Type:

STRING

Replaced Text / Final Result

 

Replace function which replaces all occurrences of text with something else.   Case does not matter, unless the final parameter is -1.

 

The Original String is limited to 4MB.

 
The Replace Text and Replace With text is limited to 64K.

 

Example:

 

 ! change all "men" to "women"

 ! ---------------------------

 InStr = 'This is a time for all good men to use iQ-XML'

 ReplaceStr = 'men'

 ReplaceWith = 'women'

 

 FinalString = XML:Replace(InStr,Replace
