XML:HTMLtoPlainText

Parameters:

*CSTRING

HTML formatted message

<STRING>

Character or characters to insert for any <STRONG> or </STRONG> tags

Return Type:

STRING

Plain Text representation of the HTML formatted message

 

Uses to translate (as best as possible) an HTML message into Plain-Text.

 

 

Example:

 

 

PlainText        Cstring(20000)

 

 CODE

 

 PlainText = XML:HTMLtoPlainText(HTMLMessage,'**')
