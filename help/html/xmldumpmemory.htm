XML:DumpMemory

Parameters:

STRING

File name to dump XML Memory too



Return Type:

Does not return a value

 

Dumps the contents of the parsed XML document in a "Queue Like" format.

 

Used for debugging (xml:DebugQueue) for sending to a text/ascii file instead of screen such as when it's a background service.

 

I personally used it to add to the GPF Hook/Handler to dump the contents of where data was causing an issue when a GPF happened.

 

 xml:DumpMemory('c:\xmldebug.txt')
