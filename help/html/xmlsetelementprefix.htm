XML:SetElementPrefix

Parameters:

STRING

Element prefix/namespace



Return Type:

Does not return a value

 

Instructs to writer to prefix all elements with the following prefix.  A colon is not needed, as the writer will automatically provide it when a prefix is set.  To turn off this value, simply call the function with a blank parameter.

 

Example:

 

 XML:SetElementPrefix('namespace')

 XML:CreateParent('Person');AddParent()

 

 XML:CreateParent('Name')

 XML:CreateAttribute('Firstname','John')

 XML:CreateAttribute('Lastname','Boy')

 XML:AddParent(1)

 

 XML:CloseParent()

 

 XML:SetElementPrefix('')

 

 

<namespace:Person>

<namespace:Name Firstname="John" Lastname="Boy"/>

</namespace:Person>
