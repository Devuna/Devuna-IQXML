SA:GetSecondsString

Parameters:

LONG

Number of Seconds



Return Type:

STRING

Return the Time as a string

 

Takes the number of seconds and returns it as a string in the form of Minutes and Seconds.   Could be used to display a short amount of time span, like how long a backup has taken.

 

 

 

! returns 1 second

TheMonth = SA:GetSecondsString(1)

 

! returns 45 seconds

TheMonth = SA:GetSecondsString(45)

 

! returns 1 minute

TheMonth = SA:GetSecondsString(60)

 

! returns 1 minute and 2 seconds

TheMonth = SA:GetSecondsString(62)

 

! returns 6 minutes and 2 seconds

TheMonth = SA:GetSecondsString(362)
