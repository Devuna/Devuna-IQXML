XML:AddParent

Parameters:

<BYTE>

True to auto-close this tag.

<STRING>

Element Value - Only valid with Auto-Close tag is set to True.



Return Type:

Does not return a value

 

Writes out Parent and Attributes.  If the function is called with the "True" parameter,  the parent will be closed with /> instead of >.

 

See also XML:CreateParent XML:AddParentBinary

 

Example:

 

XML:CreateParent('MyRootTag');XML:AddParent()

 

XML:CreateParent('CustomerTag')

XML:CreateAttribute('Lastname','Smith')

XML:AddParent(True)

 

XML:CreateParent('Warnings')

XML:CreateAttribute('Level','Warning')

XML:AddParent(True,'Things really went haywire!')

 

XML:CloseParent()

 

 

<MyRootTag>

<CustomerTag Lastname="Smith"/>

<Warnings Level="Warning">Things really went haywire!</Warnings>

</MyRootTag>
